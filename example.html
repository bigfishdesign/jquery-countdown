jquery-countdown

Default options

Call in this example:

	
	jQuery('#countdown1').countDown('01/01/2030');

Custom markup

Call in this example:

	
	jQuery('#countdown2').countDown({ endDate: '01/01/2030', templateString: '<div>{{ days }} days to go!</div>' });

Override defaults

Call in this example:

	
		jQuery.fn.countDown.defaultOptions.templateString = '<div>{{ days }} days to go!</div>';
		jQuery('#countdown3').countDown('01/02/2030');
		jQuery('#countdown4').countDown('01/03/2030');
